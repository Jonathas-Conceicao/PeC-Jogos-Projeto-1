# Project Patterns

## Name Patters
  **Everything in english**
  - Project's name      -> CamelCase
  - Folder's name       -> camelCase
  - Nodes               -> CamelCase
  - Methodos            -> snake_case
  - Signal receptors    -> _snake_case

## File's Organization
```
.                # Raiz do repositório
├── ProjectName  # .project
|	├── script   # .gd files
|	├── scene    # .tscn files
|	├── art      # image files
|	└── sound    # music files
└── arte         # Non-alocated art
    ├── ase      # Aseprite's files
	├── png      # Exported png files
	└── gif      # Exported gif files
```
